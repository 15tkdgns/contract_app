# Constract App - Node.js 프로젝트

## 설치 방법
```bash
npm install
```

## 개발 서버 실행
```bash
npm run dev
```

## 프로덕션 빌드
```bash
npm run build
```

## 주요 의존성
- react: ^18.3.1
- react-router-dom: ^7.0.2
- tesseract.js: ^5.1.1 (OCR)
- jspdf: ^2.5.2 (PDF 생성)
- html2canvas: ^1.4.1 (스크린샷)
- reactflow: ^11.11.4 (관계 그래프)
