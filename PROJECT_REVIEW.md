# Project Review: Constract App (전세사기 예방 플랫폼)

## 1. 프로젝트 개요
**Constract App**은 사회적 문제인 '전세사기'를 예방하기 위해 사용자가 업로드한 계약서와 등기부등본을 AI로 분석하여 위험 요소를 알려주는 서비스입니다.

## 2. 핵심 기능

### 🏠 스마트 문서 분석 (Smart Analysis)
- **AI OCR (OpenAI Vision API)**: 기존 Tesseract.js보다 뛰어난 인식률로 한글 계약서 텍스트 추출.
- **AI 위험도 분석 (GPT-4o-mini)**: 추출된 텍스트와 등기부 정보를 결합하여 깡통전세, 선순위 권리 과다 등 8가지 주요 사기 유형 분석.
- **위험 지수 (Risk Score)**: 종합적인 위험도를 수치와 등급(안전~심각)으로 표현.

### 📊 시각화 및 도구
- **계약 관계도 (React Flow)**: 복잡한 계약 관계를 노드와 간선으로 시각화하여 한눈에 파악.
- **계약 타임라인**: 계약 기간 및 중요 일정을 시각적으로 표시.
- **전세가율 계산기**: 매매가 대비 전세가 비율을 즉시 계산하여 위험 여부 판별.
- **체크리스트**: 계약 전/중/후 반드시 확인해야 할 항목 가이드 제공.

### 🤖 AI 상담 도구 (Chatbot)
- **실시간 상담**: OpenAI GPT 연동으로 전세 계약 관련 궁금증에 즉각 답변.
- **FAQ 폴백**: API 장애 시에도 기본적인 가이드는 제공 가능하도록 구현.

### 📄 결과 리포트
- **PDF 생성**: 분석된 결과를 정식 리포트 형태로 다운로드 가능.
- **공유 기능**: 분석 요약본을 텍스트로 생성하여 외부 공유 지원.

## 3. 기술 스택
- **Frontend**: React, Vite, CSS Modules.
- **Visualization**: React Flow.
- **AI/OCR**: OpenAI API (Vision, GPT-4o-mini), Tesseract.js (Fallback).
- **Deployment**: Vercel (Edge Functions Proxy).

## 4. 보안 및 프라이버시
- **Local Processing**: 기본적으로 클라이언트 사이드 처리를 우선하며, 사용자가 명시적으로 분석을 시작할 때만 필요한 텍스트 정보가 AI로 전송됩니다.
- **No Data Retention**: 서버에 사용자 문서를 저장하지 않는 것을 원칙으로 설계되었습니다.
