# Constract App - 전세사기 예방 웹앱

## 프로젝트 개요
사용자가 **계약서**와 **등기부등본**을 업로드하면 전세사기 위험을 분석해주는 웹 애플리케이션

## 관련 프로젝트
| 프로젝트 | 용도 | 경로 |
|---------|------|-----|
| `constract` | 발표용 대시보드 | `/root/constract` |
| `constract-app` | 전세사기 예방 웹앱 | `/root/constract-app` |

## 핵심 기능
1. **문서 업로드**
   - 계약서 (PDF/이미지)
   - 등기부등본 (PDF/이미지)

2. **자동 분석**
   - OCR로 문서 텍스트 추출
   - 계약서-등기부 정보 대조
   - 위험 요소 자동 탐지

3. **위험도 평가**
   - 전세가율 계산
   - 소유자-임대인 일치 여부
   - 근저당/선순위 권리 분석
   - 8가지 사기 유형 체크

4. **결과 제공**
   - 위험도 점수
   - 주의사항 안내
   - PDF 보고서 다운로드

## 폴더 구조
```
constract-app/
├── src/
│   ├── components/     # UI 컴포넌트
│   ├── pages/          # 페이지 (라우팅)
│   ├── services/       # OCR, 분석 로직
│   ├── utils/          # 파싱, 계산 함수
│   ├── hooks/          # 커스텀 Hooks
│   ├── assets/         # 정적 자산
│   └── styles/         # 전역 스타일
├── public/             # 정적 파일
└── docs/               # 문서
```

## 기술 스택
- **Frontend**: React + Vite
- **OCR**: Tesseract.js (브라우저 내 처리)
- **스토리지**: 없음 (1회성 분석, 클라이언트 처리)

## 개인정보 보호 방침
- 업로드된 문서는 서버에 저장되지 않음
- 모든 분석은 브라우저 내에서 처리
- 분석 완료 후 메모리에서 즉시 삭제
- 사용자 정보 수집 없음
